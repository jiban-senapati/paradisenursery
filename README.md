# Paradise Nursery - Shopping Cart Application

## Project Overview

Paradise Nursery is an e-commerce web application for an online plant shop. Users can browse houseplants, add items to cart, and manage their shopping cart.

## Features

- Landing page with hero section and "Get Started" button
- Product catalog with 3 plant categories
- Shopping cart with add/remove items and quantity controls
- Responsive design for all devices
- Redux state management

## Technologies Used

- React 18
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Vite
- Shadcn/UI

## Pages

- **Home (/)** - Landing page with hero section
- **Plants (/products)** - Product listing by category
- **Cart (/cart)** - Shopping cart management
- **About (/about)** - Company information

## Cart Functions

- `addItem()` - Add plants to cart
- `removeItem()` - Remove plants from cart
- `updateQuantity()` - Change item quantity
- `clearCart()` - Empty the cart

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
